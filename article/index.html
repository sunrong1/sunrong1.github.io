智能化环境管理系统

项目背景和痛点

传统的环境管理系统，存在以下问题：

  1. 管理人员需要手动录入数据，效率低下
  2. 管理人员需要手动处理数据，容易出错
  3. 管理人员测试过程中才能发现环境问题，效率低下
  4. 无AI技术支持，无法自动化理解和处理数据
    如何实现AI技术驱动的环境管理系统？对AI更加友好的环境管理系统？如何实现对AI的优化和集成？是我们当前的开发重点。

项目技术栈

springboot + mybatis-plus + mysql + redis + mqtt + deepseek

专利基础

个人专利占总体专利的10%。且80%的专利是非发明专利。

什么是专利

专利是指发明人或发明人授权的人对其发明的保护。专利权是一种排他权，专利权人有权禁止他人在未经许可的情况下制造、使用、销售其专利产品或使用其专利方法。

专利的分类

专利主要可以分为以下几类：

  1. 发明专利（20年）：保护发明创造的新技术方案，要求具有新颖性、创造性和实用性。
  2. 实用新型专利（10年）：保护产品的结构或机械构造的新技术方案，要求具有新颖性和实用性。
  3. 外观设计专利(10年)：保护产品的形状、图案或其结合的新设计，要求具有新颖性和美观性。

目标凡人修仙之元婴级大佬

  1. 练气期（基础巩固）：9月14日 - 10月30日（1个月）
