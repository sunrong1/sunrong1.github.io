AI Agent开发学习和路径

AI Agent是一种人工智能驱动的应用程序或服务，旨在执行特定任务或提供智能支持。它可以是聊天机器人、虚拟助手、自动化工具等，通过机器学习和自然语言处理技术，能够理解和响应用户的需求，提供相关信息、完成任务或优化流程。

开发路径

《大模型Agent智能体开发实战》路径规划

  1. 完整学习路径
    咱们Agent课程内容非常丰富，从大模型基础入门、低代码开发 Dify/coze/n8n、Agent、RAG、微调、再到商业级实战项目，是涵盖了现在大模型方向的全栈技术。结合伙伴的技术基础及学习需求，建议伙伴逐步学习并掌握以下技术模块中的核心重点知识。

阶段一：掌握大模型基础入门知识（基础必备）

本阶段核心是掌握大模型必备入门知识，这一阶段学习，无需在底层模型方面耗费过多时间，先确保把各个流程串起来，具备开发的前提。
■ 首先第一步，建议伙伴学习正课最上方的“Agent课程开始学习前 | 必知必看-课程导学”，可以对咱们的课程建立一定的认知，有助于大模型快速入门。这一部分，建议重点学习《大模型快速接入指南》。
■ 第二步，学习【Part 1】全球顶尖在线大模型接入指南，学习该章节中前面的1-5，5个视频即可。
■ 第三步，学习了解【Part 2】开源大模型部署与调用指南，学习该章节中前面的1-7，7个视频即可。
■ 伙伴刚开始学习，建议可以直接在个人电脑上，通过 调用API 就可以学习开发 Agent。如果有本地部署大模型的需求，再去详细了解大模型部署工具。

阶段二：掌握大模型Agent开发（核心）

本阶段核心是掌握热门Agent框架 以及 基于Agent底层框架进行的实战，本阶段学习结束后，伙伴能够了解不同框架的使用场景，独立部署基于框架的Agent系统。
首先学习【Part 3】大模型Agent开发基础，学习Agent基础知识。其次学习热门Agent框架，伙伴可以看到有非常多的Agent开发框架，建议优先学习这几个热门框架：LangChain -> LangGraph -> Google ADK，这几个框架适用场景是非常高的，且技术前沿性强，极力推荐大家深入学习，掌握框架底层实现及关键组件等，能够有效提升Agent开发的实战能力和技术视野。
其中，LangChain 1.0 建议必学，在此基础上学习 LangGraph，复杂场景下比如多智能体架构等，还是需要用到 LangGraph。此外，ADK 框架如果有时间建议伙伴学习，其在企业中还未完全流行开来，因此你如果能够掌握会是很大的加分项。
■ 【Part 3】大模型Agent开发基础
■ 【Part 6】LangChain1.0 体系入门
■ 【Part 7】LangGraph应用实战
■ 【Part 9】新兴Agent开发框架实战指南ADK/SDK（【ADK】1.1-12.3）

阶段三：RAG企业知识库问答与多模态拓展（核心）

本阶段核心是掌握RAG、GraphRAG与多模态RAG系统搭建，本阶段学习结束后，伙伴能够掌握GraphRAG核心机制，实现非结构数据的检索，并能构建可处理图像、PDF等复杂文档的智能检索问答系统。RAG系统其实是解决大模型"幻觉"问题的核心方案，防止大模型"瞎编"。伙伴学完以上知识后，可以按以下路径循序渐进地学习：
■ RAG入门基础：首先学习【Part 4】RAG入门&Llama-Index实战指南 这一章节中的【RAG】1.1-5.7，建立扎实的理论基础；
■ GraphRAG实战进阶：接着重点学习【Part 11】Agentic RAG 这一章节中的【GraphRAG】1.1-7.6，GraphRAG更适合现在的应用场景；
■ 多模态能力拓展：学习多模态RAG工业级案例，【Part 22】多模态RAG本地知识库问答Agent，拓展处理图像、PDF等多样化信息的能力。

阶段四：工业级智能体项目开发实战（专项案例）

该模块基于特定的热门应用场景提供完整的技术讲解及工程化项目研发落地，包括但不限于多模态RAG知识库管理、长文本生成、DeepResearch、图/音视频生成等。建议伙伴在具备一定的 Agent 和 RAG 基础之后，就可以通过小的案例进行实战，这些工业级案例都是基于特定的应用场景进行设计的，能够帮助伙伴理解技术在不同业务场景下的适配方式，将所学技术转化为实际解决方案。
Part 22 - Part 31，都是工业级智能体项目开发实战，这一部分建议伙伴可以学习 LangChain 1.0 相关案例，比如 【Part 24】数据分析Agent开发实战 和 【Part 25】文档审核Agent开发实战。

阶段五：商业智能体项目开发实战（综合应用）

学习时一定要结合具体的项目，我们的课程在每一模块的讲解中，都配有对应的实战案例。此外，课程最后还设置了几大商业级实战项目帮助大家融会贯通，这些重量级项目非常有含金量！
这一部分建议伙伴在掌握 LangGraph 开发框架与 GraphRAG 技术之后，可以直接上手复现【Part 32】智能客服Agent：AssistGen，这个项目是电商领域智能客服系统开发，使用 LangGraph 作为底层框架，涉及到的技术栈是 LangGraph + GraphRAG，建议重点学习最后一章 5.1-5.7，是完整的Multi-Agent架构产品。
此外，建议学习【Part 33】Manus通用智能体：InsightGen，该项目使用 Google ADK 作为底层框架。一定要多动手实践，除了系统学习课程内容外，还要动手复现项目案例。
■ 【Part 32】智能客服Agent：AssistGen（LangGraph + GraphRAG）
■ 【Part 33】Manus通用智能体：InsightGen（Google ADK）
阶段六：掌握 OpenClaw 核心技术（建议学习）
随着 OpenClaw 在业界的关注度持续上升，建议伙伴深度掌握 OpenClaw 的使用与核心机制。Openclaw 这类新的 Agent 产品形态，能够帮助开发者更高效地构建可交互、可扩展的智能体应用。建议伙伴有了一定的大模型基础知识之后，可以重点学习了解 OpenClaw 的核心架构设计、功能模块与实际应用案例，理解其在业务流程中的集成方式，并尝试基于自身业务场景进行开发与部署。
■ 【加餐】OpenClaw AI数字员工开发实战
■ 【加餐】OpenClaw 记忆系统开发实践
2. 快速学习路径
伙伴如果想要推动企业级项目快速落地，在这里规划一条聚焦实战的快速学习路径，建议在有了一定的大模型基础知识之后，先掌握框架原理，再通过案例实战，最后再基于框架开发商业级项目。
■ Agent 开发：建议有了一定的 AI Agent 基础认知后，优先学习 LangChain 1.0、LangGraph 框架，复杂应用现在还是 LangGraph。
■ RAG 开发：学习大模型 RAG 入门、GraphRAG 应用实战 两章节的内容。
■ 热门工业级案例：学习 Langchain 1.0 相关案例，可以看工业级案例中的 NL2SQL 案例 以及 文档审核案例；同时建议学习 多模态RAG 两个案例。
■ 商业智能体项目开发：最后通过上手复现 【Part 32】智能客服Agent：AssistGen，将完成从理论到实战的闭环，形成一套企业级解决方案。
以上学习完成后，可以再进一步学习如 Google ADK 等框架，并尝试基于此类框架开发商业级项目，比如 “Manus”通用智能体项目的实战构建，以增强对复杂系统架构的理解与工程实践能力。
