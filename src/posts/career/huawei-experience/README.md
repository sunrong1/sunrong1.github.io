#### [上海程序员通勤实录：每天 3 小时，我是如何利用的](./commute-learning-3hours.html)
*发布于 2026-03-19 | 阅读时间：8 分钟*

住在长宁中山公园，工作在青浦，每天往返 3 小时通勤。这篇文章记录我的通勤日常和时间利用方法。

**通勤路线：**
- 🏠 住所：长宁区中山公园
- 🏢 公司：青浦区（华为）
- 🚇 地铁：2 号线 → 17 号线，70 分钟
- ⏱️ 单程：1.5 小时，往返 3 小时
- 📅 每周：6 天班
- 📊 每月通勤：约 72 小时

**时间利用方法：**
1. 📖 英语学习（托业备考）
2. 🎧 技术播客
3. 📱 微信读书
4. ✍️ 文章构思
