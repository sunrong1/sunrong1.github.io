# 智能化环境管理系统

## 项目背景和痛点
传统的环境管理系统，存在以下问题：
1. 管理人员需要手动录入数据，效率低下
2. 管理人员需要手动处理数据，容易出错
3. 管理人员测试过程中才能发现环境问题，效率低下
4. 无AI技术支持，无法自动化理解和处理数据
