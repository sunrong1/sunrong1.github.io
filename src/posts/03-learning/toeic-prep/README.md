## 📖 备考资料推荐

### 必备资料 ⭐⭐⭐⭐⭐
1. **托业官方真题集** - 最接近真实考试
2. **ETS 官方指南** - 了解考试形式
3. **多邻国/扇贝单词** - 词汇积累

### 辅助资料 ⭐⭐⭐⭐
1. **YouTube 英语频道** - 听力磨耳朵
2. **BBC Learning English** - 日常英语
3. **托业备考 APP** - 碎片时间练习

### 不推荐 ❌
1. 过于老旧的题库（考试已改革）
2. 难度过高或过低的材料
3. 没有解析的练习题

## 💡 备考心得

### 听力提升关键
1. **精听 + 泛听结合** - 精听真题，泛听日常
2. **跟读练习** - 提升语感和发音
3. **关键词捕捉** - 不追求听懂每个词
4. **场景熟悉** - 商务场景词汇积累

### 阅读提升关键
1. **词汇量是基础** - 至少 6000 词汇
2. **长难句分析** - 语法基础要扎实
3. **时间管理** - 合理分配每部分时间
4. **错题总结** - 建立错题本

### 心态调整
- 不要急于求成，稳步提升
- 接受平台期，坚持就是胜利
- 模拟考试环境，适应考试节奏
- 保持平常心，发挥正常水平
