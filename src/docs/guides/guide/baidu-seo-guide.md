## 🔧 需要手动完成的步骤

### 步骤 1：申请百度统计

1. 访问 https://tongji.baidu.com/
2. 注册/登录百度账号
3. 添加网站：`sunrong.site`
4. 获取统计代码中的 ID（替换 `YOUR_BAIDU_TONGJI_ID`）

### 步骤 2：百度搜索资源平台认证

1. 访问 https://ziyuan.baidu.com/
2. 注册/登录百度账号
3. 添加网站：`https://sunrong.site`
4. 选择"HTML 标签验证"
5. 获取验证代码（替换 `YOUR_BAIDU_VERIFICATION_CODE`）

### 步骤 3：提交 sitemap

1. 登录百度搜索资源平台
2. 进入"数据引入" → "sitemap"
3. 提交 sitemap 地址：`https://sunrong.site/sitemap.xml`
4. 等待百度抓取（通常 1-7 天）

### 步骤 4：主动推送链接

每次发布新文章后，在百度搜索资源平台：
1. 进入"数据引入" → "手动提交"
2. 提交新文章 URL
3. 或使用 API 自动推送

## 📊 常见问题

### Q1: 为什么提交了还不收录？
**A:** 可能原因：
- 新站审核需要时间（通常 1-2 周）
- 内容质量不够高
- 缺少外链
- 网站权重低

### Q2: GitHub Pages 影响收录吗？
**A:** 有一定影响：
- GitHub Pages 在国内访问速度慢
- 百度对 GitHub 域名信任度较低
- 建议使用自定义域名（你已配置 ✅）

### Q3: 如何提高收录速度？
**A:** 
- 每天主动推送新链接
- 在其他平台发布相同内容（带原文链接）
- 增加高质量外链
- 保持内容更新频率
